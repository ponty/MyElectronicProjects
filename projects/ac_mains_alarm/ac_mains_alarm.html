3. AC mains alarm¶

Status: OK

This device makes alarm sound periodically when the near cable has AC signal. In debug mode it can be used as AC mains detector.

3.1. Pins¶

board pin AVR pin Arduino pin
in PC3 A3
speaker PB1 D9
debug in PC0 A0
debug LED PC5 A5
